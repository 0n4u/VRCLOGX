# VRCLOGX
Inspired by [VRCX](https://github.com/vrcx-team/VRCX) & [VRCLOG](https://github.com/ShayBox/VRC-LOG)

Before anything to get best performance add this to your launch options "--enable-debug-gui --enable-sdk-log-levels --enable-udon-debug-logging --enable-crash-dumps --enable-profile-counters --enable-fps-counter --enable-verbose-logging --no-crash-handler -logfile "C:\Logs\VRChat\vrchat_startup_%COMPUTERNAME%_%YEAR%-%MONTH%-%DAY%_%H%M%S.log"
